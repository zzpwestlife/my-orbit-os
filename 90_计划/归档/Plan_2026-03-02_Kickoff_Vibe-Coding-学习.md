## 备注

### Vibe Coding 核心要点速览

**10个实用建议:**
1. **AI First** - 遇事问 AI，培养 AI 使用习惯
2. **别在铲子上省钱** - 使用第一梯队模型 (GPT 5.3 Codex、Claude Opus 4.6、Gemini 3.0 Pro)
3. **了解模型能力边界** - 根据任务特点选择合适模型
4. **学会提问与沟通** - 使用 plan 模式，避免上下文污染
5. **管理者心态** - 划定需求边界、谋定而后动、保持质疑
6. **认识幻觉** - 提供参考资料、使用搜索工具、设定回答边界
7. **SDD (规格驱动开发)** - 消除模糊性、契约式开发、验证即终点
8. **用例子代替描述** - Few-Shot Prompting、物理约束
9. **保持 AI 清醒大脑** - 切碎任务、即用即弃、全新开局
10. **让 AI 管理代码** - 自动 Git 工作流、原子化提交
