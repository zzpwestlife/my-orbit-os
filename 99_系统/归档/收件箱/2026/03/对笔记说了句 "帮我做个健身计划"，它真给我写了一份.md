你的笔记现在在哪？

我猜：备忘录里一堆，微信收藏吃灰，Notion 开一次卡一次，飞书文档翻半天找不到。

我之前也是这样。后来用了 Obsidian，再给它接上 AI，整个人都不一样了。

今天就教你两件事，5 分钟搞定：

① 下载 Obsidian，写下第一条笔记

② 装个 AI 插件，让笔记自己 "长脑子" ✅ 提前说下，下面的截图很多不是麻烦，而是我为了让小白也能看清楚一步一步的截出来的，其实很简单，不要被吓到！

𝟭. 先把 Obsidian 装上

为啥用这个不用别的？四个字：本地、免费。

- 笔记存在你自己电脑上，不怕哪天服务跑路
- Markdown 格式，纯文本，秒开不卡
- 插件多到离谱，后面你就知道了

去 [obsidian](https://obsidian.md/) 下载，选你的系统装上就行。

![图像](https://pbs.twimg.com/media/HAthO08bMAAWANp?format=jpg&name=large)

𝟮. 建个仓库

装好打开，Obsidian 会让你选怎么开始。

点「新建仓库」，起个名字，选个文件夹，完事。

仓库就是个文件夹，没什么高级的，你所有笔记以后都在这里面。

![图像](https://pbs.twimg.com/media/HAtjvf0aAAMSB1u?format=jpg&name=large)

𝟯. 写第一条笔记

点左上角新建（或者快捷键 ⌘N），随便写点啥。

"Hello Obsidian" 也行，今天午饭吃了啥也行。

![图像](https://pbs.twimg.com/media/HAthlg7bIAAZjQJ?format=jpg&name=large)

你会发现这编辑器特别干净，没有乱七八糟的工具栏，就你和文字。

![图像](https://pbs.twimg.com/media/HAthrL3bYAAUfQy?format=jpg&name=large)

OK，Obsidian 搞定了。下面才是今天的重点。

𝟰. 给笔记接上 AI

接下来装一个叫 Claudian 的插件。

装完之后你可以直接在笔记里跟 AI 聊天，让它帮你写东西、整理内容，甚至自动管文件。听着玄乎，装完你就懂了。

过程稍微长一点，跟着截图走就行。

𝗦𝘁𝗲𝗽 𝟭: 装 Claude Code

Claudian 底层跑的是 Claude Code，先把它装上。打开终端：

```bash
npm install -g @anthropic-ai/claude-code
```

装好之后敲 which claude，能看到路径就说明成功了。记一下这个路径，待会要用。

![图像](https://pbs.twimg.com/media/HAth6XCbMAEbBI-?format=jpg&name=large)

（API Key 怎么搞？看雨哥这篇 [Antigravity Tools](https://x.com/xiangxiang103/status/2019349337143406796?s=20) 的文章就行，自己有 Claude 订阅或者用中转站也可以。）

𝗦𝘁𝗲𝗽 𝟮: 打开第三方插件

左下角齿轮 → 第三方插件 → 把安全模式关掉。

![图像](https://pbs.twimg.com/media/HAtiJqia8AAVJXu?format=jpg&name=large)

关了之后会多出来一个「浏览」按钮，点进去就是插件市场。

![图像](https://pbs.twimg.com/media/HAtiSsYaoAAkORR?format=jpg&name=large)

𝗦𝘁𝗲𝗽 𝟯: 装 BRAT

在插件市场搜 "BRAT"，装第一个（55 万人在用那个）。

![图像](https://pbs.twimg.com/media/HAtiWolasAEdGPk?format=jpg&name=large)

装完记得点「启用」。

![图像](https://pbs.twimg.com/media/HAtieHWaEAAPJ2j?format=jpg&name=large)

𝗦𝘁𝗲𝗽 𝟰: 用 BRAT 装 Claudian

回到设置页，确认 BRAT 开关打开了 ❶，然后点左边的 BRAT 进去 ❷。

![图像](https://pbs.twimg.com/media/HAtirJ6aAAAwuQs?format=jpg&name=large)

点「Add beta plugin」。

![图像](https://pbs.twimg.com/media/HAtiwu5aoAAxGKB?format=jpg&name=large)

填上这个地址：

[https://github.com/YishenTu/claudian](https://github.com/YishenTu/claudian)

点 Add Plugin，等它装完就行。

![图像](https://pbs.twimg.com/media/HAtiz3NbUAEL6fS?format=jpg&name=large)

搞定。

𝗦𝘁𝗲𝗽 𝟱: 配一下 Claudian

左边找到 Claudian 点进去，先把语言切成中文。

![图像](https://pbs.twimg.com/media/HAti3J0bQAAaz4F?format=jpg&name=large)

然后往下找到「Claude CLI 路径」，把刚才 Step 1 记的那个路径粘进去。

> 用反代或中转的话，上面「自定义变量」里填一下 API Key 和地址就行。

![图像](https://pbs.twimg.com/media/HAtjB7EaEAA3Zet?format=jpg&name=large)

𝗦𝘁𝗲𝗽 𝟲: 试一下

搞完你会发现左边多了个 🤖 图标，点它。

![图像](https://pbs.twimg.com/media/HAtjJZ6bgAAd774?format=jpg&name=large)

右边弹出来的就是 AI 面板。随便给它个任务 —— 我让它帮我做了个 7 天健身计划：

![图像](https://pbs.twimg.com/media/HAtjOrdacAAxQ-q?format=jpg&name=large)

几秒钟，一份完整的计划直接写进笔记里了：

![图像](https://pbs.twimg.com/media/HAtjR31asAAQh9o?format=jpg&name=large)

就这样，两件事做完了：装了 Obsidian，接上了 AI。

不过说实话这才刚开了个头。
